🎯 2. 포트폴리오 목적 본 포트폴리오는 대규모 클라우드 인프라의 핵심인 데이터센터 운영(DCO, Data Center Operations) 직무의 프로세스와 언어를 이해하고, 생성형 AI 및 자동화 스크립트(Python, CLI)를 활용해 현장 협업 문제를 논리적으로 해결해 나가는 학습 과정을 기록한 기술 블로그형 산출물 저장소입니다.

💡 3. 교육용 샘플 데이터 사용 안내 본 저장소에 포함된 모든 로그, 티켓, 시나리오 및 구성 요소는 실제 AWS 운영 환경의 데이터가 아닙니다.

공개된 직무 기술서(Job Description)와 교육용으로 가공된 가상 샘플 데이터(EDU, SAMPLE, DEMO)만을 바탕으로 수행한 학습용 연구 결과물입니다.

📂 4. 각 폴더의 역할 (구조 및 산출물) 01_dco_glossary/ : 데이터센터 현장에서 쓰이는 핵심 IT 인프라 및 운영 용어 정의서

02_dco_profile/ : 본인의 협업 강점(절차 준수, 소통, 체크리스트 활용 등)과 INFJ 성향을 직무 맥락에 투명하게 매칭한 협업 유형 리포트

03_sop_analysis/ : 교육용 가상 가이드라인(SAMPLE-DCO-SOP-04 등)을 분석하고 논리적 오류나 단정적 표현을 검토한 서면 분석서

04_environment_setup/ : 가상 실습을 위한 로컬 및 클라우드 개발 환경 구축 가이드

05_antigravity_cli/ : 가상 머신 및 인프라 제어를 연습하기 위한 CLI 도구 활용 환경 분석

06_cli_file_automation/ : 반복되는 인프라 관리 업무와 체크리스트 생성을 자동화하기 위한 CLI 기반 파일 자동화 스크립트

07_log_analysis_script/ : Link Down, CRC Error 등 대량의 가상 시스템 로그를 필터링하고 추이를 파악하는 Python 기초 분석 스크립트

08_incident_analysis/ : 가상의 인프라 이상 징후(과열, 부품 불량 등) 시나리오를 바탕으로 다각적인 원인을 추론한 트러블슈팅 프로세스 기록

09_incident_report/ : 가상 장애 발생 후, 원인을 단정 짓지 않고 객관적 사실 위주로 서술한 교육용 Incident Report 및 Handover Memo 초안

10_stitch_dco_dashboard/ : 교육용 인프라 상태 및 가상 티켓 진행 현황을 scannable하게 시각화하기 위한 프로토타입 대시보드 구성안

11_exam_prep/ : 인프라 지식 강화를 위해 학습한 직무 관련 인증 시험(CCNA, AWS SAA 등) 및 기술 이론 요약 노트
